# PDF-Merger-GUI

A simple and user-friendly GUI application to merge multiple PDF files into a single document.

## Features

- ✅ Simple and intuitive graphical user interface
- ✅ Merge multiple PDF files in the desired order
- ✅ Select and arrange PDF files easily
- ✅ Save the merged PDF with a custom name
- ✅ Built with Python and Tkinter

## Requirements

- Python 3.6 or higher
- PyPDF2 library
- Tkinter (usually comes with Python)

## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/rashid-alii/PDF-Merger-GUI.git
   cd PDF-Merger-GUI
   ```

2. Install required packages:
   ```bash
   pip install PyPDF2
   ```

## Usage

Run the application:
```bash
python main.py
```

### How to use:
1. Click "Browse" to select PDF files you want to merge
2. Arrange the files in the desired order
3. Click "Merge PDFs" to combine them
4. Choose a location and name for the output file
5. Your merged PDF is ready!

## Technologies Used

- Python 3
- Tkinter (GUI Framework)
- PyPDF2 (PDF Processing)
